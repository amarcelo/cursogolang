
# Desafio: Sistema Bancário em Go

## Objetivo

Você foi contratado para desenvolver um pequeno sistema bancário.

O sistema deverá possuir **dois pacotes próprios** e um programa principal (`main.go`).

Seu objetivo é organizar corretamente o projeto e aplicar o conceito de encapsulamento.

---

# Estrutura esperada

```text
banco/
│
├── go.mod
├── main.go
│
├── conta/
│   └── conta.go
│
└── calculadora/
    └── calculadora.go
```

---

# Requisitos

## Pacote calculadora

Implemente as funções:

```go
Somar()
```

```go
Subtrair()
```

```go
Multiplicar()
```

```go
Dividir()
```

A divisão deverá retornar erro caso o divisor seja zero.

---

## Pacote conta

Crie a struct:

```go
type Conta struct {
    Titular string
    saldo float64
}
```

Observe que:

* Titular deve ser público.
* saldo deve ser privado.

---

Implemente uma função construtora:

```go
NovaConta()
```

Ela deve receber:

* nome do titular
* saldo inicial

---

Crie os métodos:

```go
ConsultarSaldo()
```

```go
Depositar()
```

```go
Sacar()
```

Regras:

* não permitir depósito menor ou igual a zero;
* não permitir saque menor ou igual a zero;
* não permitir sacar mais do que existe na conta.

Todos devem retornar erro quando necessário.

---

# Programa principal

No `main.go`, faça:

## 1

Crie uma conta.

Exemplo:

```go
João
Saldo inicial: 1000
```

---

## 2

Mostre:

```text
Titular
Saldo inicial
```

---

## 3

Faça um depósito.

Exemplo:

```text
+250
```

---

## 4

Faça um saque.

Exemplo:

```text
-100
```

---

## 5

Mostre o saldo final.

---

## 6

Utilize o pacote calculadora para calcular:

```text
Saldo + 500
```

e imprima o resultado.

# Desafio Bônus 

Crie um novo pacote chamado:

```text
cliente
```

Estrutura:

```text
cliente/
    cliente.go
```

Crie a struct:

```go
type Cliente struct {
    Nome string
    CPF string
    conta conta.Conta
}
```

Observe:

* Nome é público;
* CPF é público;
* conta é privada.

Agora implemente:

```go
NovoCliente()
```

```go
Depositar()
```

```go
Sacar()
```

```go
Saldo()
```

Todos esses métodos deverão utilizar internamente a conta bancária.

O `main.go` **não pode acessar a conta diretamente**.
